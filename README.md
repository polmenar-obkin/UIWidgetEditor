# Frosty UI Widget Editor
"UI Widget Editor" is a plugin for Frosty Toolsuite which allows displaying and editing 'UIWidgetBlueprint' assets graphically, making UI modding easier in Frosty Editor.

## Features
- Real-time graphical view and editor
- Add, remove or modify UI elements of the widget easily and efficiently

## Usage
Open a UIWidgetBlueprint asset and click on the toggle button on the top-right part of the asset editor to toggle on the graph editor of the UI widget.

## Build instructions
- Clone the repository in the Frosty Toolsuite's ``Plugins`` folder
- Add this project to FrostyEditor's project dependencies
- Build & run FrostyEditor project
- You will find the generated DLL plugin in ``FrostyEditor\bin\Developer\Debug\Plugins\``
- Copy & paste the DLL inside ``Plugins`` folder of your Frosty Editor
